# МосГорБюджет.Трек

Интерактивный образовательный проект о бюджете Москвы на 2026 год. Версия подготовлена для второго тура Всероссийского конкурса «Бюджет для граждан» в номинации «Бюджет и технологии будущего».

Сайт: https://moscow-budget-dpbm.vercel.app/

> Это конкурсный прототип, а не официальный сервис Правительства Москвы. Он не связан с Mos.ID, ФНС, «Миллионом призов» или «Активным гражданином», не отправляет им данные и не начисляет реальные вознаграждения.

## Задача проекта

Бюджетные таблицы сложны для первого знакомства. Проект даёт жителю понятную точку входа: показывает ключевые параметры бюджета, предлагает учебные сценарии, объясняет налоговый вычет и закрепляет материал через задания.

## Что реализовано

- адаптивный интерфейс для компьютеров и смартфонов, светлая и тёмная темы;
- вводная экскурсия с маскотом Фини;
- учебный калькулятор социального налогового вычета со ссылкой на ФНС;
- пять тематических викторин и проверенный банк вопросов;
- пять мини-игр и симулятор распределения условного районного бюджета;
- интерактивная структура расходов и режим сравнения масштаба вычета;
- локальный справочник по заранее подготовленным темам;
- учебные баллы, уровни и прогресс, хранящиеся только в браузере.

## Граница достоверности

Верхнеуровневые показатели 2026 года основаны на официальных источниках:

- доходы — 5 937,4 млрд ₽;
- расходы — 6 385,0 млрд ₽;
- дефицит — 447,6 млрд ₽;
- социальная сфера — около 3,2 трлн ₽;
- образование — 814,6 млрд ₽;
- социальная поддержка — 810 млрд ₽;
- здравоохранение — 615 млрд ₽.

Доли подкатегорий, районные суммы, индексы и игровые результаты являются демонстрационными. Это явно отмечено в интерфейсе. Калькулятор показывает ориентир по базовой ставке 13%; фактический возврат зависит от налоговой базы, ставки и уплаченного НДФЛ.

## Технологии

React 19, TypeScript, Vite, Tailwind CSS, Motion и lucide-react. Приложение полностью клиентское: у него нет бэкенда, внешней авторизации и API-ключей. Пользовательские настройки сохраняются в `localStorage`.

## Локальный запуск

Требуется Node.js 20.19 или новее.

```bash
npm ci
npm run dev
```

Проверка перед публикацией:

```bash
npm run lint
npm run build
npm audit
```

Production-сборка создаётся в `dist/` как один HTML-файл.

## Возможное развитие

Подключение городских систем, реальных профилей или внешней программы поощрения возможно только после согласования с владельцами этих систем, правового анализа и разработки защищённого серверного контура. Текущая версия демонстрирует пользовательский сценарий без имитации действующей интеграции.
