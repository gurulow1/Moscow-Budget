# МосГорБюджет.Трек 🏛️

**Интерактивный игровой центр финансовой грамотности для портала «Открытый бюджет города Москвы»**

Конкурсный проект для номинации **«Бюджет для граждан»**. Сервис превращает изучение городского бюджета из сухой статистики в интерактивный опыт: пользователь проходит обучающий тур, играет в мини-игры, решает викторины, рассчитывает свой налоговый вычет и наглядно видит, на что и в каких долях расходуется бюджет Москвы на 2026 год.

---

## 🎯 Проблема и решение

Порталы открытых данных публикуют бюджет в виде таблиц и сводок, которые сложны для восприятия обычным гражданином. Из-за этого падает вовлечённость и теряется главная цель открытого бюджета — **прозрачность и понятность для жителя**.

**МосГорБюджет.Трек** решает это за счёт геймификации:

- сложные бюджетные понятия объясняются через игру и интерактив;
- житель видит **личный вклад** своего НДФЛ в конкретные программы города;
- механики баллов и уровней удерживают внимание и мотивируют изучать материал до конца;
- концептуальная интеграция с городскими экосистемами **«Миллион призов»** и **«Активный Гражданин»** превращает обучение в реальную пользу.

## ✨ Ключевые возможности

| Раздел | Что внутри |
| --- | --- |
| 🎓 **Обучающий тур** | Пошаговый онбординг с гидом-маскотом Фини — проводит по всем разделам портала |
| 🧮 **Калькулятор 3-НДФЛ** | Расчёт социального налогового вычета за обучение и спорт (лимит 150 000 ₽, возврат 13%) |
| ❓ **Викторины** | 53 вопроса по финансовой грамотности и устройству бюджета |
| 🎮 **5 мини-игр** | Балансировщик бюджета, Финансовый Аудитор, Инвест-Стратег, Вычет-Клик, Вектор Развития |
| 👑 **Симулятор «Мэр округа»** | Распределение бюджетных квот района и защита проекта бюджета |
| 🗺️ **Карта округов** | Интерактивный гид по административным округам Москвы |
| 📊 **Аналитика расходов** | Структура расходов-2026 с подкатегориями и режимом «Мой вклад» |
| 🤖 **ИИ-ассистент** | Бюджетный помощник на базе локальной базы знаний (41 статья), работает без внешних ключей |
| 🏆 **Геймификация** | Баллы, опыт (XP), уровни, прогресс — с концептом обмена в экосистемах города |

Дополнительно: адаптивная вёрстка (ПК и смартфон), тёмная тема, доступность (фокус-кольца для навигации с клавиатуры).

## 📐 Достоверность данных

Все ключевые цифры выверены по официальным параметрам **Закона города Москвы «О бюджете города Москвы на 2026 год и плановый период 2027 и 2028 годов»** (принят 1 ноября 2025 г.):

- доходы 2026 — **5,93 трлн ₽**, расходы — **6,38 трлн ₽**, дефицит — **≈447 млрд ₽**;
- бюджет формируется на основе **13 государственных программ** (>90% расходов);
- транспорт — 1,29 трлн ₽, образование — 814,6 млрд ₽, соцподдержка (госпрограмма) — 810 млрд ₽, здравоохранение — 615 млрд ₽;
- на социальную сферу города в целом направлено около половины бюджета (≈3,2 трлн ₽).

**Источники:**
- Портал «Открытый бюджет города Москвы» — https://budget.mos.ru/
- Государственные программы Москвы — https://budget.mos.ru/budget/gp
- Мосгордума приняла бюджет на 2026–2028 годы — https://www.interfax.ru/moscow/1055935

> Цифры по округам в разделе «Карта» носят иллюстративно-игровой характер и служат для демонстрации механики.

## 🛠️ Технологии

- **React 19** + **TypeScript**
- **Vite 6** (сборка и dev-сервер)
- **Tailwind CSS v4** (стилизация)
- **Motion** (анимации), **Recharts** (графики), **Three.js** (фон), **lucide-react** (иконки)

Приложение полностью клиентское: работает в браузере без бэкенда и без API-ключей. Прогресс пользователя сохраняется локально (localStorage).

## 🚀 Запуск

Требуется **Node.js 18+**.

```bash
cd app
npm install      # установка зависимостей
npm run dev      # запуск в режиме разработки (http://localhost:3000)
npm run build    # production-сборка в папку dist/
npm run preview  # просмотр production-сборки
```
